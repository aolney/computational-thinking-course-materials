Measurement to Fractals

Our second class introduced recursion and fractals. Our final exercise for that week was making a fractal fern.

That fern had "made-up" branching angles and branch lengths. This week we will use measurements of branch angle and length. That means we will be working with data. The image below is a reminder of what the data labels mean.

This is a multi-step exercise. Do the following in order:

  • Get one data block, Mean, StandardDeviation, ColumnData, and GetParameter blocks.
  • Fill in the missing components of the GetParameter block using the other functions.
  • Call GetParameter a couple of times with different variable names.
  • Stop here for discussion.
  • Get and run the Fern block and refamiliarize yourself with how it works.
  • Get the Shrub function, and using the Fern as a guide, fill in the GetParameter blocks.
  • Get the shrub parameter block (speed Infinity) and Shrub run block, and run.
  • Stop here for discussion.
  • Get the while block, put it inside Shrub after `cumulativeLength=0`, change GetParameter, and run it.
  • Stop here for discussion.
  • Get the angle/if block, put it inside the while after `fd internodeLength * 3`, change GetParameter, and run it.
  • Stop here for discussion.
  • Get the second Shrub run block and data block and do an additional Shrub run. You should see the two shrubs side by side.
