Breeder Systems: Face

We're going to breed faces.

This lab loads more data over the internet than previous labs. You may notice a 5-10 second delay every time you run the final program.

This is a multi-step exercise. Do the following in order:

  • Get Ex 1 block. This block sets global evolution parameters.
    • Set parameters genomesize, popsize, and mutation rate to 50, 18, and .2
  • Get Ex 2 block. This block defines how the population is initially randomized and how mutation happens.
    • Set allele range in random genome and set allele range in mutate to 256.
  • Get Ex 3 block. This block defines how children are produced from parents.
    • Fill in blank to implement crossover.
  • Get Ex 4 block. This block sets up the breeder system, UI, and matching of genotypes (arrays) to phenotypes (faces).
    • Fill in blank in ClosestFace.
  • Cloning: Pick a face you'll remember. Click it twice. On the next round, click it again. Keep doing this for 10-20 rounds. What happens?
  • Breeding
    • Restart the program.
    • Choose two different criteria about the face (male/female, attractiveness, facial feature shape, skin tone, etc.)
    • Try to breed faces so that all faces in the working set have the first criteria you selected.
    • Restart the program.
    • Try to breed faces so that all faces in the working set have the second criteria you selected.
